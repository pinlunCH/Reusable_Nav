# Reusable code
## Show & Hide Navigation

Need to be modify:
* nav-toggle(Class Name)
* standard-nav(Class Name)
* nav-close(Class Name)
* put in num(style.right number)

CSS
```
{
.nav-toggle.off{
    display: none;
}
.nav-toggle{}
.nav-close{
    display: none;
}
.nav-close.active{
    display: block;
}
```
